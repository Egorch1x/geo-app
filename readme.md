# GeoQuiz — Образовательный тренажер по географии

**GeoQuiz** — это веб-приложение для изучения столиц стран мира. Проект создан в образовательных целях.

## Функционал
- Просмотр списка всех добавленных стран.
- Админ-панель для добавления новых стран (с валидацией данных).
- Интерактивная викторина: угадай столицу по названию страны.
- Сообщения об ошибках и успехах (Flash-уведомления).

## Технологии
- **Backend:** Python (Flask)
- **Database:** SQLAlchemy (SQLite)
- **Frontend:** Jinja2, Tailwind CSS
- **Forms:** Flask-WTF

## Инструкция по запуску
1. Клонируйте репозиторий:
   `git clone https://github.com/Egorch1x/geo-app`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Запустите приложение:
   `python app.py`
4. Откройте в браузере: `http://127.0.0.1:5000`
